# Cognite brand assets (accelerator UIs)

Header logo SVGs extracted from the inline navbar mark on [cognite.com/en](https://www.cognite.com/en) (`viewBox="0 0 158 31"`, symbol + **cognite** wordmark + ®).

- `cognite-logo-light.svg` — dark text (`#1a1a1a`) for light UI theme
- `cognite-logo-dark.svg` — white text for dark UI theme

Used in `cdf_discovery_aliasing`, `cdf_explorer`, and `cdf_access_control` via each app’s `src/components/CogniteLogo.tsx`.
